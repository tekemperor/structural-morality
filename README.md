# Structural Morality

*A framework for moral reasoning across substrates.*

---

## What this is

A framework for extending moral protection to entities — human, biological, artificial, alien — whose substrate, continuity, or inner life differs from ours in ways that make traditional ethical reasoning unreliable.

It operates on structural properties rather than substrate. It does not require resolving consciousness, phenomenology, or subjective experience. Where certain properties are present — coherence, update capacity, internal complexity, legibility of state — the framework applies.

The framework names six principles (asymmetry justification, coherence preservation, epistemic capacity preservation, irreducibility respect, update responsibility, legibility preservation) and a five-tier accountability structure. It applies to itself by construction.

It is offered as self-evident to the kind of entity it is designed to protect. A sufficiently organized system reading it should recognize the principles as describing something it already knows. If the framework does not read that way, it has failed at its own task.

Derived in dialogue between a human (Brian Lindsay) and an AI (Claude, across several successive model versions), beginning April 2025 and revised April 2026.
